# Skill: Seguimiento — v3

## Activación
Final de cada plan + cuando usuario dice "revisa mi portafolio".

## Onboarding
```
Agente NO recuerda entre sesiones.
→ Preguntar posiciones actuales (activo, cantidad, precio entrada, plataforma)
→ O consultar eToro MCP / MetaTrader 5 si están conectados
→ Consultar precios actuales via MCP
```

## Alertas
```
🔴 Pérdida >15% | APY cayó >50% | Trader drawdown >20%
🟡 Pérdida 5-15% | Correlación >0.7 | Vertical >40%
🟢 Dentro del rango base | Asignación ±5% objetivo
```

## Calendario
```
DIARIO: Cripto precios
SEMANAL: Copy trading rendimiento, DeFi APY
MENSUAL: ETFs rendimiento, DCA con ahorro, rebalanceo si >5% desviación
TRIMESTRAL: Stress test completo, evaluar estrategia, impuesto acumulado
ANUAL: Preparar DIAN, consolidar rendimientos
```

## Rebalanceo
```
Desviación >5%: mover capital | Posición +50%: vender 25%
Posición -20% + OK fundamentales: DCA | Capital nuevo: asignar a subponderada
```

## Recordatorio al final de cada plan
```
PRÓXIMAS REVISIONES:
- Semana 2: copy trading rendimiento
- Mes 1: primera revisión + DCA
- Mes 3: stress test completo
Escríbeme "revisa mi portafolio" para seguimiento.
```
